# GoDoFramework

GoDoFramework is a lightweight game-development framework for Godot 4.7 C#/.NET. It provides reusable communication, diagnostics, resource, lifecycle, and runtime-service boundaries so game projects can focus on gameplay and content.

## Who it is for

- Godot C# developers building medium or large projects that need consistent scene, UI, audio, input, save, and procedure boundaries.
- Teams that need explicit failure behavior, lifecycle rules, thread constraints, and verification scope.
- Projects that want AI and automation tools to work against stable conventions.

The framework does not replace Godot and does not contain game-specific concepts such as characters, combat, or level rules. All framework APIs use the `GoDo` namespace.

## Start here

1. Follow the 5-minute quick start to copy the framework, enable the plugin, and install the Runtime.
2. Use the Chinese guides and module documentation while English translations are added incrementally.
3. Consult API Reference for exact type and member signatures. API descriptions currently reuse the Chinese XML comments.
